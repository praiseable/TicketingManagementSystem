# UI Quality Pack 01 — Dashboard + Project Settings

This pack improves UAT readiness by adding frontend validation, feedback, loading states, confirmation prompts, and an operational dashboard.

## Implemented

- Dashboard ownership and assignment visibility.
- Shared API error helper.
- Shared feedback components.
- Shared loading button component.
- Improved confirmation dialog.
- Project Settings validation for members, issue types, custom fields, workflows, statuses, transitions, and guards.
- Manual test cases in `docs/test-cases/UI_QUALITY_PACK_01_TEST_CASES.md`.
- Static smoke checks in `scripts/smoke-ui-quality-pack-01.sh`.
